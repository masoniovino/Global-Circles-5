# Global Circles — Multiplayer

## Setup
```
npm install
node server.js
```
Then open http://localhost:3000 in multiple browser tabs or share with friends on your network via http://YOUR_IP:3000

## Deploy online (free)
- **Railway**: Push to GitHub, connect repo at railway.app, set start command to `node server.js`
- **Render**: New Web Service → connect repo → `node server.js`
- **Fly.io**: `fly launch` then `fly deploy`

The PORT environment variable is respected automatically.
